## 🎨 设计理念

### 核心原则

1. **色彩和谐** - 每个主题都经过精心调色，确保色彩搭配协调
2. **视觉层次** - 清晰的明暗对比，建立信息层级
3. **舒适阅读** - 优化的对比度，长时间使用不疲劳
4. **现代美学** - 采用当代设计趋势，简洁优雅
5. **一致性** - 统一的圆角、间距、阴影系统

## 🎯 设计细节

### 1. 色彩系统

#### 背景层次 (3层深度)
```
Layer 1 (Primary):   最深 - 主背景
Layer 2 (Secondary): 中等 - 侧边栏/头部
Layer 3 (Card):      最浅 - 卡片/内容区
```

**示例 (紫罗兰主题)**:
- Primary: `#1a1025` (最深)
- Secondary: `#231735` (中等)
- Card: `#2d1f45` (最浅)

这种设计创造了自然的视觉深度，引导用户注意力。

---

#### 文字层次 (3级对比)
```
Primary:   85-90% 亮度 - 主要内容
Secondary: 60-70% 亮度 - 次要信息
Muted:     40-50% 亮度 - 辅助文本
```

**示例 (深空灰主题)**:
- Primary: `#e5e5e5` (90%)
- Secondary: `#a3a3a3` (64%)
- Muted: `#737373` (45%)

确保良好的可读性和视觉层次。

---

#### 强调色系统
```
Primary Accent:   主操作、链接、激活状态
Secondary Accent: 成功、确认、辅助操作
Warning:          警告、注意
Error:            错误、危险
Purple:           特殊功能、高级选项
```

**统一标准**:
- Warning: `#fbbf24` (所有主题一致)
- Error: `#f87171` (所有主题一致)
- Purple: `#a78bfa` 或 `#c084fc` (根据主题调整)

### 3. 阴影系统

#### 发光阴影 (Glow)
```css
shadowGlow: '0 0 24px rgba(color, intensity)'
```

**强度分级**:
- 低 (0.1-0.15): 暗夜科技、纯黑OLED
- 中 (0.18-0.2): 深空灰、翡翠绿、琥珀橙、玫瑰粉
- 高 (0.22): 紫罗兰

#### 卡片阴影 (Card)
```css
shadowCard: '0 8px 32px rgba(0, 0, 0, depth)'
```

**深度分级**:
- 浅 (0.5-0.6): 深色主题
- 中 (0.6): 大多数主题
- 深 (0.8): 纯黑OLED

## 🎨 配色技巧

### 1. HSL 色彩模型

所有主题都基于 HSL 模型设计：

**紫罗兰主题示例**:
```
Hue: 270° (紫色)
Saturation: 60-80% (适中饱和)
Lightness: 
  - Background: 8-15% (深)
  - Text: 85-90% (亮)
  - Accent: 65-75% (鲜艳)
```

这种方法确保色彩的和谐统一。

---

### 2. 对比度检查

所有主题的文字对比度都符合 **WCAG AA 标准**：

- 正常文字: ≥ 4.5:1
- 大文字: ≥ 3:1
- UI组件: ≥ 3:1

## 🔧 技术实现

### CSS 变量映射

```typescript
// themes.ts 中的配置
colors: {
  bgPrimary: '#0d1117',      // → --bg-primary
  accentPrimary: '#58a6ff',  // → --accent-primary
  borderRadius: '8px',       // → --radius
  // ...
}

// theme.css 中的应用
background: var(--bg-primary);
color: var(--accent-primary);
border-radius: var(--radius);
```

---

### 主题切换动画

```typescript
document.body.style.transition = 'all 0.3s ease'
setTimeout(() => {
  document.body.style.transition = ''
}, 300)
```

平滑过渡所有颜色变化，避免闪烁。

## ✨ 设计亮点

### 1. 渐变色预览
每个主题的预览都使用渐变色：
```tsx
background: `linear-gradient(135deg, ${accentPrimary}, ${accentSecondary})`
```

直观展示主题的两种主色调。

---

### 2. 智能发光
根据主题色自动调整发光颜色：
```css
boxShadow: currentTheme === key 
  ? `0 0 20px ${theme.colors.accentPrimary}40` 
  : 'none'
```
